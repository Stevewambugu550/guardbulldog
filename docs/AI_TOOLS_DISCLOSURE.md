# AI Tools Disclosure

## GuardBulldog Project - AI Usage Statement

This document discloses the use of AI tools during the development of the GuardBulldog Campus Phishing Detection & Response Platform and the writing of the technical report.

---

## 1. AI Tools Used in Project Development

### 1.1 Code Development & Assistance
| Tool | Purpose | Usage |
|------|---------|-------|
| **Windsurf Cascade** | AI coding assistant | Used for code generation, debugging, refactoring, and implementing features across the React frontend and Node.js backend |
| **GitHub Copilot** | Code completion | Assisted with code suggestions and autocompletion during development |

### 1.2 Specific Development Tasks Assisted by AI
- **Frontend Development**: React component creation, TypeScript type definitions, UI/UX implementation
- **Backend Development**: Express.js route handlers, middleware implementation, database queries
- **Database Design**: Schema design assistance, SQL query optimization
- **API Development**: RESTful endpoint design, request/response handling
- **Authentication**: Implementation of authentication flows and security middleware
- **Documentation**: Code comments, README files, and technical documentation

---

## 2. AI Tools Used in Technical Report Writing

### 2.1 Documentation Assistance
| Tool | Purpose | Usage |
|------|---------|-------|
| **Windsurf Cascade** | Documentation generation | Assisted in creating technical diagrams (ERD, System Architecture), formatting documentation |

### 2.2 Specific Documentation Tasks
- **Diagram Creation**: ERD diagrams, system architecture diagrams, sequence diagrams
- **Technical Writing**: Formatting and structuring technical documentation
- **Markdown Formatting**: Proper formatting of documentation files

---

## 3. Human Oversight & Verification

All AI-generated content was:
1. **Reviewed** by the development team for accuracy and correctness
2. **Tested** to ensure functionality meets requirements
3. **Modified** as needed to align with project specifications
4. **Validated** against security best practices

---

## 4. Original Work Statement

While AI tools were used to assist in development and documentation:
- The **project concept and design** are original work
- All **architectural decisions** were made by the development team
- **Business logic and requirements** were defined by the team
- AI tools served as **productivity enhancers**, not replacements for human judgment

---

## 5. Ethical Considerations

- AI tools were used in compliance with academic integrity policies
- All AI usage is transparently disclosed in this document
- The development team maintains full responsibility for the final product
